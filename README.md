# Cybersecurity Port Scanner Project

## 📌 Project Description
This is a GUI-based Network Port Scanner developed using Python. It scans a target system to identify open ports and detect potential vulnerabilities.

## 🎯 Features
- Scan open ports in a given range
- Displays system IP address
- User-friendly GUI using Tkinter
- Real-time scanning output

## 🛠 Technologies Used
- Python
- Socket Programming
- Tkinter (GUI)

## ▶️ How to Run
1. Open the project file in Python IDLE
2. Run the program
3. Enter:
   - Target IP (example: 127.0.0.1)
   - Port range (1–100)
4. Click "Start Scan"
